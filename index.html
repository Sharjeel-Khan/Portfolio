Research Topics

Detecting and mitigating side-channel attacks at runtime

Georgia Institute of Technology

We are trying to detect side-channel attacks at runtime with our Beacon framework.

September 2019 - Present

Predicting Call Chain using Sequence to Sequence Model

Georgia Institute of Technology

We built an LLVM intermediate representation (IR) pass which produces the call chain for each program. We are using Tensorflow's sequence to sequence model to take a section of the call chain and use it to predict the next function in call chain. Using this model, our scheduler can try to schedule programs based on the next function of each program.

September 2019 - Present

Predicting Dynamic Program Paths using Decision Trees

Georgia Institute of Technology

We built an LLVM intermediate representation (IR) pass which creates a path profile for the entire program. Using the path profile and a trained decision tree classifier on path profile, we are trying to predict the dynamic path early in the program with a high accuracy alongside a minimum overhead. The early prediction allows us to check if an attacker changes the path by either control or non-control data attacks.

September 2018 - Present

metaCLF2

Carnegie Mellon University

We researched about financial systems more specifically Automated Trading Systems (ATS). We formalized ATS in Celf, a concurrent logic framework (CLF), so we can prove financial properties like no crossed/locked market. In addition, we built the meta-theory in Celf, so Celf could verify these properties automatically.
